X2Go is an open source remote desktop software for Linux that uses the NX technology protocol.

## Installation

ffmpeg is quickly installed through the command:

```bash main
sudo box install x2go
```

## Client Side
```plaintext
Session Name: This is an identifier for the server.
Host: Domain or IP Address
Username: Your SSH/Swizzin Username
Port: 22
Session Type: XFCE
```

After clicking OK, you should get a block on the right side of your screen with the details of the configuration. Click on this and you will be greeted with a login screen for you to type in your password.
