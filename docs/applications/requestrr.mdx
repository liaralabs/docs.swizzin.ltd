Requestrr is a Requestrr is a chatbot used to simplify using services like Sonarr/Radarr/Ombi via the use of chat!

## Initial Setup

Installing Requestrr is easy. Simply issue the following command from SSH:

```bash main
sudo box install requestrr
```
This command will configure and install Requestrr for your user.

After installation, the config is stored in `/opt/requestrr/config/`.

## How to Access

After installation, you can access the web client at `https://<hostname.ltd>/Requestrr/`.

## Configuration & Usage

Requestrr needs to be confiugred to be integrated through discord and the *arrs.

## Connecting the Bot to Discord
1. Go to Chat Clients
2. Once youre there you'll need to go to the Discord Developer Panel [Here](https://discordapp.com/developers/applications)
3. Login and Create an Application
4. Fill out the Name for your new application
5. Copy the `Client ID`
6. Paste the Client ID into Requestrr
7. Go back to the Discord Developer Portal and Create a Bot for your Application
8. Change the username if you desire and give your bot a cool profile picture!
9. Turn on `Presence Intent` and `Server Members Intent`
10. Now copy the Bot Token
11. Paste it into Requestrr in the Bot Token Section
12. Click Test Settings button and it should say everything is good
13. Click Save Changes ath the bottom of the page

## Inviting Bot to the Server and Testing
1. Copy the Invite Link
2. Paste it into your browser and choose your Discord Server
3. Test your new bot out by typing `ping!` in chat and it should respond `!pong`
